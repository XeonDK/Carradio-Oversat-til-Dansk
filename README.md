# Instillation

1. Download den

2. Åben den

3. gå ind på dine filer

4. gå ind i Resources 

5. Smid den ind

6. gå ind i din server.cfg eller server_resources.cfg kommer and på hvad filer du bruger!

7. Start den

8. og så Genstart din Server:D
